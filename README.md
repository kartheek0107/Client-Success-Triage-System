# Client-Success-Triage-System
An AI-powered ticket classification system to prioritize client issues, ensuring critical business-impacting problems get addressed faster.

## Technical Approach

NLP models (spaCy, scikit-learn, HuggingFace Transformers)

FastAPI service for ticket intake & classification

Data storage in MongoDB

Dashboard built with Streamlit

## Expected Impact

Cut response time for high-priority tickets.

Improve client satisfaction through smarter triage routing

Handle 10k+ support tickets/day with scalable pipeline
